# Getting Started

> Quick reminder that this is an **experimental project**.

## How to collect data

To collect data via a custom form (which is the only way right now), send a _POST_ request that looks like following:

```tsx
fetch("https://fast-forward.app/api/feedback", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    projectId: "...", // mandatory
    text: comment, // mandatory
    category: "", // mandatory - either "issue", "idea" or "other"
    userId: user.email, // optional
    metadata: {}, // optional - e.g. { lang: "en", position: "header" }
  }),
});
```

If you are interested in how I created the floating _Widget_ form, feel free to check out the [GitHub repository](https://github.com/maximiliankaske/fast-forward/blob/main/components/widget/WidgetFABExample.tsx). Following libraries have been used:

- [tailwindcss](https://tailwindcss.com/) for **styling**
- [headlessui](https://headlessui.dev/) for **Dialog (Modal)** Box
- [pooper](https://popper.js.org/) for **automatic placement** of the Dialog Box depending on the space
